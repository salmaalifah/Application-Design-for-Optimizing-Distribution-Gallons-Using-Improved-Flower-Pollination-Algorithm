# Application Design for Optimizing Distribution Gallons Using Improved Flower Pollination Algorithm

I build an optimization model that use Improved Flower Pollination Algorithm in solving provlems of non-optimal delivery route. This website use MySQL for database, PHP, CSS, Javascript, HTML for language programming, Cpanel for hosting, and Google Cloud Platform for Library Maps API. The results of the application show that using the IFPA method produces a more optimal route than using the company's usual route. The detail of dataset is consist of 50 customers (name, address, latitude, longitude).
